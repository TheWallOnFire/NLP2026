# app

This folder contains the application code for the NLP2026 project.

## Overview

The `app` directory is intended to hold the executable application, modules, and utilities used to run the NLP pipeline or web service for this project.

## Structure

- `app/` - Main application package or script files
- `README.md` - This documentation file

## Usage

1. Ensure dependencies are installed from the project requirements.
2. Run the main application entrypoint from the `app` folder, for example:
   ```bash
   python -m app
   ```
3. Consult the individual module docstrings for details on available functionality.

## Notes

- Keep the `app` folder focused on runtime code and application logic.
- Separate data processing, model training, and utilities into appropriate subpackages if needed.
